# Business Intelligence Insights: Sales & Profit Analysis

## 🎯 Project Objectives
The goal of this dashboard is to provide actionable insights into retail performance by answering the following key business questions:

1.  **Product Performance:** Identify the Top and Bottom 5 products based on Sales, Profit, and Quantity Sold.
2.  **Temporal Trends:** Analyze how sales fluctuate across different time grains (Daily, Monthly, Quarterly, and Annually).
3.  **Profitability Correlation:** Visualize the direct relationship between Sales and Profit to identify high-margin opportunities.
4.  **Comparative Analysis:** Enable users to compare Sales, Profit, and Quantity between any two custom-selected time periods.
5.  **Discount Impact:** Calculate the average discount offered within each specific discount category.
6.  **Operational Volume:** Track the total number of orders processed.
7.  **Granular Data Exploration:** A detailed drill-through table showing Sales, Profit, Discount, and Net Sales for every order, filterable by Product, Date, Customer, and Promotion.
8.  **Geographic Distribution:** Map out sales performance across different cities.

## 🚀 Key Features
* **Time Intelligence:** Advanced DAX used for Period-over-Period (PoP) comparisons.
* **Dynamic Visuals:** Interactive filters for Customer ID and Promotion Categories.
* **Geospatial Mapping:** Visualization of regional sales density.

## 🛠️ Tools Used
* **Power BI Desktop:** Dashboarding and Data Visualization.
* **Power Query:** Data transformation and cleaning.
* **DAX (Data Analysis Expressions):** Calculated measures for complex KPIs.
